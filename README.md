<h1 align="center">👁️ Drishti – Speech to Indian Sign Language Conversion</h1>

<p align="center">
An AI-powered assistive communication system that converts live speech into Indian Sign Language (ISL) gestures in real time.
</p>

---

## 📌 Overview

Drishti is an intelligent communication system designed to help hearing-impaired individuals understand spoken language through Indian Sign Language (ISL).

The user speaks through a microphone, and the system recognizes the speech and converts it into corresponding ISL signs or gestures in real time.

The project aims to bridge the communication gap between normal speakers and the deaf community using AI and assistive technologies.

---

## ✨ Features

- 🎤 Real-Time Speech Recognition
- 🤟 Speech to ISL Conversion
- 🧠 AI-Based Language Processing
- ⚡ Instant Gesture Output
- 🖥️ User-Friendly Interface
- ♿ Assistive Communication System

---

## 🛠️ Technologies Used

- Python
- Speech Recognition
- OpenCV
- Natural Language Processing (NLP)
- Machine Learning

---

## ⚙️ Working Principle

- User gives live speech input
- Speech recognition converts audio into text
- NLP processes the recognized text
- Corresponding ISL gestures are generated
- ISL output is displayed in real time

---

## 🚀 How to Run

### Step 1 — Install Dependencies

```bash
pip install opencv-python speechrecognition
```

### Step 2 — Run the Project

```bash
python drishti.py
```

<h1 align="center">🗣️ Dhvani – Sign Language to Speech Conversion</h1>

<p align="center">
An AI-powered assistive communication system that converts sign language gestures into spoken words or sentences in real time.
</p>

---

## 📌 Overview

Dhvani is an intelligent assistive system designed to help speech and hearing-impaired individuals communicate effectively with others.

The user performs sign language gestures in front of a camera, and the system recognizes the gestures using computer vision and machine learning techniques. The recognized signs are then converted into speech output.

---

## ✨ Features

- 🤟 Real-Time Sign Language Recognition
- 🎥 Camera-Based Gesture Detection
- 🔊 Gesture to Speech Conversion
- 🧠 AI & Computer Vision Integration
- ⚡ Instant Voice Output
- 🖥️ Interactive User Interface

---

## 🛠️ Technologies Used

- Python
- OpenCV
- MediaPipe
- Machine Learning
- Text-to-Speech (TTS)

---

## ⚙️ Working Principle

- User shows sign language gestures
- Camera captures hand movements
- AI model recognizes gestures
- Recognized text is generated
- Text is converted into speech output

---

## 🚀 How to Run

### Step 1 — Install Dependencies

```bash
pip install opencv-python mediapipe pyttsx3
```

### Step 2 — Run the Project

```bash
python dhvani.py
```
